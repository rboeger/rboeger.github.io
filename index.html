Ping Pong Game

This is a pong game inspired by Super Ping Pong for the Sega Genesis. Loaded with inside jokes because I play it with friends on game nights. I had a lot of fun making powerups and making ps5 controller support. Powerups include: Bounce(Horizontal and Vertical), speed change(increase and decrease), skull(if ball hits paddle, it will lose the round), multiple balls, shield, random location, change paddle size, and sticky paddle.
